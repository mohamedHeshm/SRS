## 🚀 خطوات البدء (5 دقائق فقط!)

### 1️⃣ إعداد Supabase
```bash
1. اذهب إلى https://supabase.com
2. أنشئ مشروع جديد
3. اذهب إلى Settings → API
4. انسخ URL و Anon Key
```

### 2️⃣ تحديث script.js
```javascript
// ابحث عن هذه الأسطر:
const SUPABASE_URL = 'YOUR-URL-HERE';
const SUPABASE_ANON_KEY = 'YOUR-KEY-HERE';

// ثم استبدل بقيمك
```

### 3️⃣ إنشاء قاعدة البيانات
```bash
1. فتح SQL Editor في Supabase
2. انسخ محتوى database.sql
3. الصق والضغط على "Run"
```

### 4️⃣ تشغيل المنصة
```bash
# Option 1: Python
python -m http.server 3000

# Option 2: Node.js
http-server -p 3000

# Option 3: VS Code Live Server
Click Right → Open with Live Server
```

### 5️⃣ الاختبار
```
1. افتح http://localhost:3000
2. اضغط "ابدأ مجانًا"
3. أنشئ حساب
4. استمتع! 🎉
```
